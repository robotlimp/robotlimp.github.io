Verifiably Following Complex Robot Instructions with Foundation Models

Language Instruction grounding for Motion Planning (LIMP) executing the instruction "Bring the green plush toy to the whiteboard in front of it, watch out for the robot in front of the toy". LIMP grounds instruction referents and generates a plan that enables the robot to navigate from its start location (yellow, A), to the green plush toy (green, B), execute a pick skill which searches for and grasps the object, then navigate to the whiteboard (blue, C), and execute a place skill which sets the object down.
